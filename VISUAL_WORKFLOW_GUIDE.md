## Placeholder System

### Format
```
{{placeholder_name}}
```

### Examples
```html
<!-- Simple text replacement -->
<p>This certifies that {{student_name}} has completed {{course_name}}</p>

<!-- In headings -->
<h1>Certificate for {{student_name}}</h1>

<!-- In tables -->
<table>
  <tr>
    <td>Name:</td>
    <td>{{student_name}}</td>
  </tr>
  <tr>
    <td>Course:</td>
    <td>{{course_name}}</td>
  </tr>
</table>

<!-- With styling -->
<span style="color: blue; font-size: 24px;">{{student_name}}</span>
```

### Extraction Process
```
Input HTML:
  "Certificate for {{student_name}} in {{course_name}}"

Regex: /{{(.*?)}}/g

Matches:
  1. {{student_name}}
  2. {{course_name}}

Extracted:
  ["student_name", "course_name"]

Stored in DB:
  placeholders: '["student_name", "course_name"]'
```

## API Request/Response Examples

### Create Template
```http
POST /api/templates
Authorization: Bearer {jwt_token}
Content-Type: application/json

{
  "name": "Course Completion Certificate",
  "description": "Standard certificate for courses",
  "templateContent": "<h1>Certificate for {{student_name}}</h1>",
  "type": "HTML"
}

Response:
{
  "success": true,
  "message": "Template created successfully",
  "data": {
    "id": 1,
    "name": "Course Completion Certificate",
    "placeholders": ["student_name"],
    ...
  }
}
```

### Simulate Template
```http
POST /api/templates/simulate
Authorization: Bearer {jwt_token}
Content-Type: application/json

{
  "templateId": 1,
  "placeholderValues": {
    "student_name": "John Doe",
    "course_name": "Web Development"
  }
}

Response:
{
  "success": true,
  "data": {
    "previewHtml": "<h1>Certificate for John Doe</h1>",
    "extractedPlaceholders": ["student_name", "course_name"],
    "success": true
  }
}
```
